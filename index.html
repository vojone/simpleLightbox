Simplest usage

Just add "lb" class to photos, that you want to have in galery. For caption use data-caption attribute in HTML


Multiple galeries

If you want to have multiple galeries just add subclass after "lb" (important is only first subclass name after "lb"). Images with same subclass should be in the same galery.
